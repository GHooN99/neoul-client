## Neoul client
> 모두의 다이어리, 너울 (제작 중)

### 프로젝트 설명 
자발적인 학습을 통하여 알게 된 다양한 기술을 적용해보고자 시작한 개인 프로젝트입니다.

### Tech Stack
`react v18` `next v12`  `typescript` `styled-components` `react-query` `storybook` `etc...`
